## iMA 理解锁定句

```text
我先按[模式]处理，只看[范围]，先给你[详细度]结果。
```

## iMA 首轮极简回复

```markdown
判断：____
范围：____
结论：____

回复 `1继续`、`2展开`、`3换模式` 或 `4只结论` 即可。
```

## iMA 知识库任务首轮

```markdown
我先按 Plan Mode 处理，只看最相关的 3 条材料，先给你标准结果。

判断：这是知识库任务。
范围：先不读全部资料，只抓最相关内容。
下一步：你可回复 `1继续`、`2展开`、`5只列材料`。
```

## iMA 模糊任务首轮

```markdown
这个任务我先不展开。你选一个：

1. 只给结论
2. 先列材料
3. 先给计划
4. 完整展开

默认建议：1 或 3。
```
