# Example: Expert Escalation

## User request

`这个前端页面一直错乱，修了三次还不行。`

## Route

- clarity: medium
- complexity: medium to complex
- risk: medium
- token budget: Standard
- mode: Expert Mode recommendation

## Sample response

```markdown
## 建议召唤专家

当前问题已超出普通快速处理范围，继续试错可能造成 token 浪费。

## 建议专家类型

资深前端工程师

## 需要专家解决的问题

1. 页面布局错乱的根因定位。
2. 响应式规则、组件样式和状态驱动逻辑的冲突排查。
```
