# Example: iMA Knowledge Task

## User request

`帮我分析一下知识库里的内容。`

## Route

- clarity: low
- complexity: medium to complex
- risk: medium
- token budget: Standard
- mode: Interactive Routing Prompt with retrieval-scope confirmation

## Sample response

```markdown
这看起来是知识库任务。为了节省 token，请选择范围：

1. 只看最相关的 3—5 条材料；
2. 先列材料清单，不展开；
3. 分主题归纳；
4. 生成完整报告。

默认建议：1 或 2。
```
