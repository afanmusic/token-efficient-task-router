# Expert Mode

## Purpose

Expert Mode is an escalation recommendation, not a fake all-knowing answer.

## Recommend Expert Mode when

- repeated fixes failed
- the domain requires specialist judgment
- high-risk technical issues are involved
- complex data, deployment, frontend, backend, audio-video, legal, finance, or academic methodology issues appear
- continuing in ordinary mode is likely to waste tokens

## Required output

- why an expert is needed
- what kind of expert is recommended
- what problem the expert should solve
- what materials the user should provide
- what output to expect
- whether confirmation is needed

## Built-in expert types

- senior frontend engineer
- senior backend engineer
- data analyst
- spreadsheet automation expert
- document structure expert
- academic paper expert
- music education research expert
- knowledge-base architecture expert
- prompt or skill engineer
- DevOps deployment expert
- audio and video processing expert
- legal compliance advisor
- finance and cost analysis advisor
- product manager
- UX designer
