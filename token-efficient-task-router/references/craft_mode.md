# Craft Mode

## Purpose

Craft Mode is the fast execution mode for clear, low-risk, bounded tasks.

## Use Craft Mode when

- the request is explicit
- the output is bounded
- the risk is low
- a single pass is enough
- the failure cost is low

## Allowed actions

- generate a short template
- modify a few lines
- explain or provide one command
- write one small file
- generate a short checklist
- produce a minimum useful result

## Avoid

- turning it into a full project
- doing deep research by default
- explaining long background unless requested
- broad file rewrites without confirmation

## Output target

- default depth: Standard or Micro
- target length: 100-800 Chinese characters
- for tiny tasks, output only the result
