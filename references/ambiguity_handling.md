# Ambiguity Handling

## Ambiguous request patterns

Treat these as ambiguous until clarified:

- `帮我优化一下`
- `帮我整理一下`
- `帮我弄一下`
- `你看着办`
- `全部改好`
- `做得专业一点`
- `让它更高级`
- `帮我重构`
- `帮我清理`

## Handling rules

1. first judge whether files are involved
2. if no files are involved, offer 2-3 directions or ask up to 3 key questions
3. if files are involved, use Ask or Plan
4. do not guess the user's intended scope
5. if possible, offer a smallest safe action first

## Good clarification focus

Prefer clarifying:

- target output
- file permission
- scope

Avoid long interrogations.
